Achieve AI is a mobile-first goal achievement app that uses AI to create personalized, step-by-step action plans. Users set goals, and the AI generates detailed instructions to help them achieve those goals.

üì± How to Run

Development

  1. Open terminal/command prompt
  2. Navigate to project: cd C:\Users\talme\achieve-ai
  3. Start server: python -m http.server 8080
  4. Open browser: http://localhost:8080/simple.html

Testing on Mobile

  1. Find your computer's IP address
  2. Make sure phone is on same WiFi
  3. Open: http://YOUR_IP:8080/simple.html
