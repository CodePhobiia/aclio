# App Store Privacy Labels - Aclio

Use this document when filling out the App Privacy section in App Store Connect.

## Data Collection Summary

### Does this app collect data?
**Yes**

---

## Data Types Collected

### 1. Identifiers
- **Device ID**
  - Used for: App Functionality
  - Linked to Identity: No
  - Used for Tracking: No

### 2. Usage Data
- **Product Interaction**
  - Used for: App Functionality, Analytics
  - Linked to Identity: No
  - Used for Tracking: No

### 3. Location
- **Coarse Location** (City-level only)
  - Used for: App Functionality (personalized recommendations)
  - Linked to Identity: No
  - Used for Tracking: No
  - Note: Only collected when user explicitly enables in Settings

### 4. Purchases
- **Purchase History**
  - Used for: App Functionality (subscription management)
  - Linked to Identity: No
  - Used for Tracking: No

### 5. User Content
- **Other User Content** (Goals, chat messages)
  - Used for: App Functionality (AI-powered planning)
  - Linked to Identity: No
  - Used for Tracking: No
  - Note: Sent to OpenAI GPT API for processing, not stored permanently

---

- ❌ Contact Info (Name, Email, Phone)

- ❌ Audio

- ❌ Search History

## Third-Party SDKs

### RevenueCat
- **Purpose**: In-app purchase and subscription management
- **Data Collected**: Purchase history, device identifiers
- **Privacy Policy**: https://www.revenuecat.com/privacy

### OpenAI (GPT API)
- **Purpose**: AI-powered goal planning, step generation, chat features
- **Data Sent**: Goal descriptions, user preferences, chat messages
- **Data Retention**: Not stored permanently (processed and discarded)
- **Privacy Policy**: https://openai.com/policies/privacy-policy
